DEFINICIONES

Web
Es un sistema interconectado de páginas web públicas accesibles a través de Internet
HTML
Lenguaje estándar fundamental para estructurar y dar forma al contenido de las páginas web, definiendo la jerarquía y disposición de textos, imágenes, enlaces y otros elementos, actuando como el "esqueleto" que los navegadores interpretan para mostrar un sitio web.
XML
lenguaje de marcado que permite estructurar, almacenar y transportar datos de forma legible tanto para humanos como para máquinas
