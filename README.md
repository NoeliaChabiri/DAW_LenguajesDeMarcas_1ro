Este repositorio está dedicado a las **actividades que realizo en Lenguajes de Marcas** durante mi carrera.  
This repository is dedicated to the **activities I complete in Markup Languages** during my **first year** of study.

---

## Objetivo / Purpose

- Guardar y organizar **mis proyectos y prácticas** de HTML, CSS y otros lenguajes de marcas.  
- Store and organize **my projects and practice exercises** in HTML, CSS, and other markup languages.

- Registrar mi **progreso a lo largo del tiempo**.  
- Track my **progress over time**.

- Servir como referencia personal para futuros proyectos y estudios.  
- Serve as a personal reference for future projects and coursework.

---

## Estructura del repositorio / Repository Structure

- **Carpeta1 / Folder1**: Primeras prácticas y ejercicios básicos / Initial exercises and basic practice.  
- **Carpeta2 / Folder2**: Actividades más avanzadas y ejercicios combinados / More advanced activities and combined exercises.  
- **Carpeta3 / Folder3**: Proyectos personales y experimentos / Personal projects and experiments.

---

## Uso / Usage

- Cada archivo HTML se puede abrir directamente en un navegador para ver los resultados.  
- Each HTML file can be opened directly in a browser to see the results.

- Los archivos CSS asociados están dentro de cada carpeta y se vinculan a los archivos HTML correspondientes.  
- The associated CSS files are located inside each folder and linked to the corresponding HTML files.
